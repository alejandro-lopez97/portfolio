# Portfolio Website

A clean, professional portfolio website with smart and weighted design. No flashy elements - just focused, professional presentation.

## Features

- **About Me**: Personal introduction and background
- **Expertise**: Technical skills and competencies  
- **Experience**: Professional work history
- **Projects**: Showcase of notable work
- **Contact**: LinkedIn, Gmail, Resume links in navbar

## Design Philosophy

- Smart and weighted styling - no unnecessary elements
- Clean, readable typography
- Professional color scheme
- Fast loading and accessible
- Mobile-responsive design

## Tech Stack

- HTML5, CSS3, JavaScript
- CSS Grid and Flexbox for layout
- Google Fonts for typography
- Font Awesome for icons
- Mobile-first responsive design

## Setup

1. Open index.html in browser
2. Customize content sections
3. Add your images and resume
4. Update contact links
5. Deploy to hosting platform

## Customization

Replace placeholder content with:
- Your professional information
- Project details and screenshots
- Resume PDF file
- Contact information
- Profile photo
